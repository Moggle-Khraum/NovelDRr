## 📖 About

NovelDR is a free, open-source Android application that lets you download webnovels from multiple sources and read them offline. Built for readers who want uninterrupted access to their favorite stories without ads, accounts, or internet dependency.

## ✨ Features

| Category | Details |
|---|---|
| **🌐 Multi-Source Download** | ReadNovelFull, NovelFull, FreeWebNovel, NovelBin, LightNovelWorld, & more to come |
| **📚 Smart Library** | Filter by All/Unread/Reading/Completed, batch delete, long-press actions |
| **🔄 One-Tap Updates** | Select a novel and tap Update to fetch new chapters |
| **🎨 3 Reading Themes** | Dark, Light, and Sepia modes |
| **📖 Reading Customization** | Adjustable font size, line spacing, and auto-scroll |
| **💾 Data Control** | Full File Backup , Restore & Import system |
| **⏭️ Smart Skipper** | Skip existing chapters when downloading to avoid duplicates |
| **🔒 Fully Offline** | Everything stored locally — no account or internet needed after download |

## 🚀 Quick Install Guide

1. Download the APK from [Releases](https://github.com/Moggle-Khraum/noveldr-site/releases) or from [Website](https://moggle.is-a-good.dev/#)
2. Open your file manager and locate the `.apk` file
3. Tap to install (allow "Unknown Sources" if prompted)
4. Open Novel DR
5. Paste a novel URL and start downloading!

**Looking for older versions?** Browse the [Archives](https://github.com/Moggle-Khraum/noveldr-site/releases).
